> Definí una función `gano` que reciba como argumento un array `tragamonedas` :slot_machine: con 5 símbolos y nos indique si son iguales. Si el array tiene más de 5 símbolos, sólo debe comparar los 5 primeros.
>
```javascript
ム gano(['⭐ '⭐️', '⭐️', '💫', '✨']) // 
false
ム gano(['💫', '💫', '💫', '💫', '💫']) 
true
ム gano(['💫', '💫', '💫', '💫', '💫', '⭐️']) 
true
```
